## The three decisions that gate everything else
1. **Strips: proprietary vs commercial?** (TPP §4) — changes scope, dossier, and timeline most.
2. **Confirm CDSCO class via pre-submission** before generating validation data.
3. **Keep or drop UV-C sterilization?** (Risk H1 / Standards §UV-C) — highest novelty-vs-burden tradeoff.
